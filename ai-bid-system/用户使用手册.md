## 1. 注册与登录

### 注册新账号

1. 打开系统网址（通常是http://192.168.169.107:5173/）
2. 在登录页面点击 **"注册"** 链接
3. 填写以下信息：
   - **姓名**：你的真实姓名
   - **邮箱**：有效邮箱地址（用于密码找回）
   - **密码**：设置安全密码
   - **确认密码**：再次输入密码
4. 点击 **"注册"** 按钮
5. 注册成功后会自动登录并跳转到主页

### 登录系统

1. 打开系统网址
2. 输入注册时使用的 **邮箱** 和 **密码**（默认使用账号wayneli12345@gmail.com  密码：qwe123123 已预存信息）
3. 点击 **"登录"** 按钮
4. 登录成功后进入系统主页

### 忘记密码

1. 在登录页面点击 **"忘记密码"** 链接
2. 输入注册邮箱
3. 按提示完成密码重置

> 💡 **提示**：登录后建议先去 **个人资料** 页面完善姓名和所属公司信息。

## 3. 招标解读 — 上传招标文件获取分析报告

当你拿到一份招标文件，想快速了解招标要求时，使用此功能。

### 操作步骤

1. 在左侧菜单点击 **"招标解读"**
2. 进入页面后，你会看到一个文件上传区域
3. **上传招标文件**：
   - 将文件拖拽到上传区域，或点击上传区域选择文件
   - 支持格式：`.docx`、`.pdf`
   - 文件大小限制：10MB
4. 上传后系统会自动开始分析，进度条会显示处理进度
5. **等待分析完成**（通常需要1-3分钟，请勿关闭页面）
6. 分析完成后，你会看到三份报告：
   - **招标文件解读**：项目基本信息、资格审查要求、评标标准
   - **招标框架提取**：投标文件应包含的结构和格式
   - **商务资料清单**：需要准备的资质证明、文件材料清单
7. 点击 **"下载报告"** 可将分析结果导出

### 分析报告包含什么？

| 报告部分 | 内容 |
|---------|------|
| 基础审核 | 项目名称、编号、招标人、预算金额、关键时间节点 |
| 资格与合规 | 营业执照、财务状况、社保要求、特殊资质 |
| 评标与计分 | 价格分、技术分、商务分的权重和评分标准 |
| 废标项排查 | 会导致投标无效的条款，标注高风险项 |
| 商务资料清单 | 所有需要准备的材料清单及要求 |
| 总体建议 | 是否建议投标、需要准备的核心材料 |

> ⚠️ **注意**：分析过程中请保持页面打开，关闭页面会导致分析中断。

## 4. 新建标书 — 智能生成投标文档

这是系统的核心功能，通过AI帮你快速生成和填充投标文档。

### 操作步骤

#### 第一步：选择投标主体

1. 在左侧菜单点击 **"新建标书"**
2. 系统会提示你选择一个投标主体（公司）
   - 如果公司信息库中已有数据，直接从列表选择
   - 如果还没有，点击 **"新建公司"** 先去公司信息库添加
3. 选择公司后点击 **"确认"**

#### 第二步：上传招标文件

1. 点击上传区域或拖拽文件到上传区
2. 支持格式：`.docx`
3. 上传后系统会自动扫描文档中的空白项和表格

#### 第三步：扫描空白与表格

上传文件后，系统自动执行以下操作：
- **扫描空白**：识别文档中所有需要填写的位置（如下划线、空格、括号等）
- **识别表格**：将文档中的表格分为三类：
  - 普通填空：简单的文字空白处
  - 动态表格：可复用的人员简历表、项目经历表等
  - 人工表格：报价表、偏离表等涉及金额的高风险表格

#### 第四步：智能填充

系统会根据你资源库中的数据（公司信息、产品资料、人员信息等），自动填充识别到的空白：

1. **自动填充**：AI自动匹配并填入相关数据（如公司名称、地址、法人等）
2. **手动调整**：你可以逐项检查并修改AI填充的内容
3. **选择数据源**：对于每个空白，可以选择不同的数据来源

#### 第五步：审核与导出

1. 检查所有填充内容是否正确
2. 系统会进行 **智能审核**，标注可能的问题
3. 确认无误后，点击 **"导出"** 按钮
4. 系统会生成完整的 `.docx` 投标文档并自动下载

### 填充时的技巧

- 📌 **公司信息库越完善，自动填充越准确**：提前在公司信息库中录入完整的公司资料
- 📌 **产品资产库帮助填写技术参数**：产品信息越详细，技术部分填充越好
- 📌 **人员专家库用于表格填充**：简历表、项目经历表会自动从人员库拉取数据
- 📌 **涉及金额的表格请务必人工核对**：报价表、偏离表等AI不会自动填充，需要手动确认

## 6. 公司信息库 — 管理投标主体信息

录入你公司的基本信息，这些数据会用于标书自动填充。

### 操作步骤

1. 在左侧菜单点击 **"公司信息库"**
2. 点击 **"新建主体"** 按钮
3. 填写公司信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 公司名称 | 公司全称 | XX科技有限公司 |
| 统一社会信用代码 | 18位代码 | 91110108... |
| 注册资金 | 如 1000万元 | 1000万元 |
| 公司性质 | 企业类型 | 有限责任公司 |
| 成立日期 | 注册日期 | 2015-01-01 |
| 经营期限 | 有效期 | 2015-01-01 至 2055-01-01 |
| 联系电话 | 公司电话 | 010-12345678 |
| 公司邮箱 | 对外邮箱 | contact@xx.com |
| 公司地址 | 注册地址 | 北京市XX区XX路XX号 |
| 邮政编码 | 邮编 | 100000 |
| 登记机关 | 工商局 | 北京市XX区市场监督管理局 |

1. 在表单底部点击 **"添加自定义字段"**
2. 输入字段名称和值
3. 保存即可

> 💡 **提示**：建议在第一次投标前就把公司信息库完整填写好，这样创建标书时AI可以用这些数据自动填充。

## 7. 人员专家库 — 管理团队与专家信息

管理投标中需要用到的团队成员和专家信息，用于填充标书中的人员简历表、项目经历表等。

### 操作步骤

1. 在左侧菜单点击 **"人员专家库"**
2. 点击 **"新建人员"** 按钮
3. 填写人员信息：

| 字段 | 说明 |
|------|------|
| 姓名 | 人员真实姓名 |
| 性别 | 男 / 女 |
| 手机号 | 联系电话 |
| 学历 | 博士 / 硕士 / 本科 / 大专 / 中专 / 高中 / 其他 |
| 学位 | 博士 / 硕士 / 学士 / 无学位 |
| 毕业院校 | 学校名称 |
| 专业 | 所学专业 |
| 职称 | 如 高级工程师 |
| 拟任职务 | 在本项目中的角色，如 项目经理 |

4. **上传附件**：
   - 身份证正面（必）
   - 身份证反面（必）
   - 学位证书（最多4个文件）
   - 资质证书（最多4个文件）

5. **添加项目经历**：
   - 点击 **"添加项目经历"**
   - 填写：项目名称、项目角色、开始/结束时间、工作内容描述

6. 点击 **"保存"**

> 💡 **提示**：人员信息越详细，在标书中填充简历表和项目经历表越方便。

## 8. 产品资产库 — 管理产品与服务信息

按公司分类管理产品和相关资料，用于标书中的技术参数填充和产品描述。

### 操作步骤

1. 在左侧菜单点击 **"产品资产库"**
2. 页面左侧显示 **产品树形结构**，按公司 → 产品的层级组织

#### 添加新产品

1. 点击 **"新增产品"** 按钮
2. 填写：
   - **所属公司**：选择或新建公司名
   - **产品名称**：如 "XX型号路由器"
   - **版本号**（选填）：如 "V2.0"
3. 保存后产品出现在左侧树形列表中

#### 为产品添加资产

选中一个产品后，可以在右侧为其添加 **资产**（资料）：

1. 点击 **"添加资产"**
2. 选择资产类型：
   - **图片**：上传产品图片、技术架构图等
   - **文档**：上传 PDF、Word 等产品说明文档
   - **文字**：直接输入文字内容（如技术参数描述）
3. 给资产起名并保存

> 💡 **提示**：在新建标书时，AI会从产品资产库中匹配相关产品的资料来填充标书。

## 11. 历史标书整理 — 让AI学习过往标书

上传以前做过（成功或失败）的标书，让AI学习其中的经验和模板，用于提升新标书的生成质量。

### 操作步骤

1. 在左侧菜单点击 **"历史标书整理"**
2. 点击 **"上传历史标书"** 或拖拽文件到上传区域
3. 系统会自动解析文档，提取关键信息
4. 解析完成后，你可以看到以下信息：
   - **提取的字段**：文档中的各个关键信息点
   - **学习到的模板**：文档的结构模板
   - **示例内容**：每个字段的示例值

### 学习结果的作用

| 学习内容 | 如何帮助你 |
|---------|-----------|
| 文档结构模板 | 新建标书时可以复用历史标书的结构 |
| 字段填写示例 | AI会参考历史标书中的写法来填写内容 |
| 成功经验 | 提高自动填充的准确率和专业度 |

> 💡 **提示**：上传越多高质量的历史标书，AI生成的效果越好。建议至少上传2-3份成功中标的历史标书。

## 13. 推荐工作流程

为了获得最好的投标效果，建议按以下顺序操作：

```
第一步：完善基础资料
  ├── 进入"公司信息库"，录入公司基本信息和法人信息
  ├── 进入"人员专家库"，录入项目团队和专家信息  
  ├── 进入"产品资产库"，录入产品信息和技术参数
  ├── 进入"图片库"，上传资质证书、项目照片等
  └── 进入"知识库"，上传常用的投标文档和模板

第二步（可选）：让AI学习经验
  └── 进入"历史标书整理"，上传2-3份过往成功标书

第三步：分析招标文件
  └── 进入"招标解读"，上传招标文件，获取AI分析报告

第四步：生成投标文档
  └── 进入"新建标书"，选择投标主体 → 上传招标文件 → AI自动填充 → 审核 → 导出

第五步：管理与迭代
  └── 进入"我的标书"，查看和管理所有项目
```

### 一句话总结

> **资料越全，AI越强！** — 在开始投标前，先把公司信息库、人员专家库、产品资产库的信息填完整，AI就能更准确地帮你自动填充标书。

## 14. 常见问题

### Q：上传招标文件后分析一直没反应？
检查网络连接是否正常，刷新页面重试。如果问题持续存在，请联系管理员确认后端服务和Dify AI是否正常运行。

### Q：智能填充的内容不准确怎么办？
AI填充的内容是基于你资源库中的数据进行匹配的。请在以下方面检查：
- 公司信息库中的信息是否完整和准确
- 产品资产库中是否录入了相关产品
- 人员专家库中是否有对应的人员信息
- 知识库中是否上传了足够的参考资料

你可以随时手动修改AI填充的内容，不必完全依赖自动填充。

### Q：上传文件提示格式不支持？
目前系统支持的文件格式如下：
- 招标解读：`.docx`、`.pdf`
- 新建标书：`.docx`
- 知识库文档：PDF、Word 等常见格式
- 图片库：JPG、PNG 等常见图片格式

请确保上传的文件格式正确，文件大小不超过 10MB。

### Q：涉及金额的表格为什么没有自动填充？
报价表、偏离表等涉及金额的表格属于高风险内容，系统设计为不自动填充，需要你手动确认和填写。这是为了确保金额数据的准确性。

### Q：如何让AI生成更专业的标书？
1. 在知识库中上传更多行业相关的参考资料
2. 在历史标书整理中上传更多成功案例
3. 在公司信息库中完善所有字段（包括自定义字段）
4. 在产品资产库中添加详细的产品说明和技术参数
