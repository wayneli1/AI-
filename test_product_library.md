## 4. 故障排除

### 问题 1：树形目录不显示
- 检查数据库连接
- 确认 `products` 表中有当前用户的数据
- 检查用户 ID 是否正确

### 问题 2：TreeSelect 不加载数据
- 确认 `targetCompany` 输入框有值
- 检查网络请求是否成功
- 确认 `products` 表中有匹配 `company_name` 的数据

### 问题 3：资产上传失败
- 检查 Supabase Storage 权限
- 确认文件大小不超过限制
- 检查文件类型是否支持

### 问题 4：Dify 提示词格式错误
- 检查 `handleAutoFill` 函数中的资产分组逻辑
- 确认占位符格式：`{{IMG_产品版本_资产名称}}`
- 检查文本内容截断逻辑
